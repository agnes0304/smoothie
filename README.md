# SMOOTHIE

안녕하세요, 스무디 마이페이지 담당 프론트엔드 엔지니어 최지우입니다.


스무디는 프로그래밍을 이제 막 시작한, 뉴비들을 위한 프로젝트 플랫폼입니다.<br/>
자신이 참여하고픈 프로젝트를 어필하고, 직접 기획한 프로젝트에 팀원을 모집할 수 있는 서비스이며, <br/>
사이트 내에서 서로에게 남긴 코멘트가 레퍼런스가 되어 자신과 잘 맞는 팀원을 모집할 수 있도록 기획했습니다.
